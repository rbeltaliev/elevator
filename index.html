Программа не для идеи

Elevator — это не старт бизнеса с нуля. Это разгон уже существующего продукта.

✕Программа для идеи
✕Программа для презентации
✕Программа для студентов
✓ Подаваться могут только проекты с готовым MVP

Во время подачи заявки обязательно прикладывается:

●Ссылка на работающий MVP
●Демо продукта
●GitHub (если применимо)
●Видео демонстрация
Без MVP заявка не рассматривается.
